## Windows

如果您正在使用 Windows 系统作为运行或者开发环境，很抱歉我们对此并没有足够的支持。如果您希望在一台 Windows 系统中搭建环境并运行 Contentos 主链，最好的方式是使用一个运行 Ubuntu 系统的虚拟机，并在虚拟机中完成相应的工作。
